# C Library

> یک کتابخانه استاندارد و کاربردی در زبان C شامل توابع ریاضی و رشته‌ای برای مصارف عمومی

---

## 📚 فهرست توابع

### توابع ریاضی

| تابع | توضیحات |
|------|---------|
| `isPrime(int n)` | بررسی عدد اول بودن |
| `factorial(int n)` | محاسبه فاکتوریل |
| `reverseNumber(int n)` | معکوس کردن عدد |
| `sumDigits(int n)` | محاسبه جمع ارقام |
| `countDigits(int n)` | شمارش تعداد ارقام |
| `gcd(int a, int b)` | بزرگترین مقسوم‌علیه مشترک |
| `lcm(int a, int b)` | کوچکترین مضرب مشترک |
| `power(int base, int exp)` | محاسبه توان |
| `isEven(int n)` | بررسی زوج بودن |

### توابع رشته‌ای

| تابع | توضیحات |
|------|---------|
| `reverseString(char str[])` | معکوس کردن رشته |
